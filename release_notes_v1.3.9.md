## Panduan Instalasi (Split APK)

Kami menyediakan 3 jenis file APK yang berbeda berdasarkan arsitektur prosesor perangkat Anda. Hal ini dilakukan untuk **memperkecil ukuran file instalasi**. Silakan pilih salah satu yang sesuai:

1.  **`app-arm64-v8a-release.apk` (Rekomendasi)**
    *   **Untuk apa?** Perangkat Android modern (64-bit).
    *   **Kapan digunakan?** Jika HP Anda keluaran tahun 2017 ke atas, kemungkinan besar menggunakan arsitektur ini. Ini adalah pilihan terbaik untuk performa maksimal.

2.  **`app-armeabi-v7a-release.apk`**
    *   **Untuk apa?** Perangkat Android lama (32-bit).
    *   **Kapan digunakan?** Jika HP Anda adalah model lama (budget phone lama) yang tidak mendukung 64-bit.

3.  **`app-x86_64-release.apk`**
    *   **Untuk apa?** Emulator Android di PC atau tablet berbasis Intel/AMD.
    *   **Kapan digunakan?** Jika Anda menjalankan aplikasi ini di laptop/PC menggunakan emulator seperti BlueStacks, LDPlayer, atau emulator Android Studio.

### Cara Memasang:
1.  Unduh salah satu file di atas sesuai perangkat Anda.
2.  Buka file APK tersebut di HP Anda.
3.  Izinkan "Install dari sumber tidak dikenal" jika diminta.
4.  Selamat menabung! 💰
