# Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi di **TabunganKu**! Kami sangat terbuka bagi siapa pun yang ingin membantu mengembangkan aplikasi ini menjadi lebih baik.

## Cara Berkontribusi

1. **Fork** repository ini.
2. Buat **Branch** baru untuk fitur atau perbaikan Anda (`git checkout -b fitur/nama-fitur`).
3. Lakukan **Commit** dengan pesan yang jelas (`git commit -m "Menambah fitur scan struk"`).
4. **Push** ke branch tersebut (`git push origin fitur/nama-fitur`).
5. Buat **Pull Request** (PR).

## Standar Kode

- Pastikan kode Anda mengikuti [Standard Flutter Linting](https://pub.dev/packages/flutter_lints).
- Dokumentasikan fungsi-fungsi penting jika perubahan Anda signifikan.
- Gunakan **Riverpod** untuk manajemen *state* (sesuai arsitektur yang sudah ada).

## Melaporkan Bug

Jika Anda menemukan bug teknis (bukan keamanan), mohon buka **Issue** di GitHub dengan format:
- Deskripsi singkat bug.
- Langkah-langkah untuk mereproduksi bug.
- Screenshot atau rekaman (jika ada).
- Versi Flutter/OS yang digunakan.
